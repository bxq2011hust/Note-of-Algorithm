## 题目1
输入一个整数，输出其二进制表示中1的个数

### 实现
```cpp
int NumberOf1(int n)
{
    int count = 0;
    while (n)
    {
        ++count;
        n &= (n - 1);
    }
    return count;
}
``` 
## 题目2
判断一个数是不是2的整数次方

**思路**：对任意的正数`n&(n-1)==0`则其是2的整数次方。
